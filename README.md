# Privacy Policy – EM Electronics

Last updated: 2026

EM Electronics d.o.o. respects the privacy of users and is committed to protecting personal data.

## Data Collection

The EM Electronics mobile application may collect the following data to enable core functionality:

- User login information (username or email)
- Device location to record start and end of work orders
- Photos that users attach to work orders as documentation

## How We Use Data

Collected data is used solely for:

- user authentication
- managing work orders
- documenting service interventions
- recording work locations

## Data Sharing

The application does not sell or share personal data with third parties. Data may be stored using secure cloud infrastructure (Firebase).

## Security

We implement reasonable technical and organizational measures to protect user data.
